# Portfolio Website
The **Portfolio Website** serves as a personal site to highlight my work, including detailed project pages, technical skills, and contact information. It is optimized for both desktop and mobile platforms, offering a seamless experience across devices.

## 🛠️ Features

- 🌟 **Project Showcase** – Explore featured works with live links and detailed project descriptions.
- 💼 **Skills Section** – Displays expertise across front-end and back-end technologies.
- 📱 **Responsive Design** – Mobile-first approach ensuring smooth display on all devices.
- 📬 **Contact Form** – Directly send messages using the form (email integration supported).
- ✨ **Custom Animations** – Smooth transitions and interactive hover effects.
- 🧩 **Interactive UI Components** – Buttons, modals, and hover effects built with modern CSS/JS.

## ⚙️ Installation

To run the portfolio locally:

```bash
# Clone the repository
git clone https://github.com/GokulakrishnanSivalingam/portpolio-2.git

# Navigate to the project directory
cd portpolio-2

# Install dependencies
npm install

# Run the development server
npm run dev
```

> Make sure **Node.js** and **npm** are installed on your machine.
